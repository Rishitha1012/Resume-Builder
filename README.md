# Resume Builder

A simple Resume Builder web application that allows users to create a resume using a clean and user-friendly interface.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User-friendly form
- Dynamic resume generation
- Clean UI design
- Easy to use

## How to Use
1. Open the project in a browser
2. Enter your details
3. Generate your resume

## Future Enhancements
- Download resume as PDF
- Multiple templates
- Improved styling
